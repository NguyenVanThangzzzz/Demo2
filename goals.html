Mục tiêu và Kỳ vọng

Mục tiêu học tập

Back-end Development

  • Nắm vững kiến trúc và nguyên lý hoạt động của Node.js
  • Thành thạo Express.js framework và RESTful API development
  • Hiểu sâu về C# và .NET Core, bao gồm Dependency Injection và Middleware

Database Management

  • Thành thạo MongoDB: Schema design, Aggregation, Indexing
  • Nắm vững MySQL: Query optimization, Transaction management
  • Hiểu và áp dụng được cả SQL và NoSQL trong các use case khác nhau

Kỳ vọng phát triển

Ngắn hạn (3-6 tháng)

  • Xây dựng được các API RESTful với Node.js và .NET
  • Thực hiện được các tác vụ CRUD với MongoDB và MySQL
  • Tham gia phát triển ít nhất 2 dự án thực tế

Dài hạn (6-12 tháng)

  • Phát triển các ứng dụng enterprise-level
  • Tối ưu hóa hiệu suất database và application
  • Đạt chứng chỉ về .NET và Node.js
